Welcome! This repository contains my [[https://grtcdr.github.io/data/resume.pdf][personal résumé]].

** You might be asking...
*** Why LaTeX?
Plain text is my jam.

*** Can I use it?
Yes, you may.

** Usage

Turn it into a PDF:

#+begin_src sh
pdflatex -halt-on-error cv.tex
#+end_src

An optional step is to optimize it to achieve a smaller size, install ~ghostscript~ beforehand:

#+begin_src sh
gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.5 -dNOPAUSE -dQUIET -dBATCH -dPrinted=false -sOutputFile=resume.min.pdf resume.pdf
#+end_src

> Q: Why LaTeX?

> A: Plain text is my jam.

> Q: Why split up the sections into different files?

> A: Structure. I can neither stand nor comprehend long LaTeX documents.

> Q: Can I use it?

> A: Yes, you may.

** Usage

Turn it into a PDF:

#+begin_src sh
pdflatex -halt-on-error cv.tex
#+end_src

Optimize it to achieve a smaller size, install ~ghostscript~ beforehand:

#+begin_src sh
gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.5 -dNOPAUSE -dQUIET -dBATCH -dPrinted=false -sOutputFile=cv.min.pdf cv.pdf
#+end_src
